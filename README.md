# GitHub Activity Generator

A project designed to generate real, meaningful GitHub activity such as documentation updates, project statistics generation, and changelogs.

## Setup

1. Clone the repository.
2. Ensure you have Python installed.
3. Install dependencies from `requirements.txt`.
4. Run the scripts in the `src/` directory or let the GitHub Actions workflow run them automatically.
